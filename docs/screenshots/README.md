# Screenshots

Add screenshots here for the README. Recommended files:

1. **dashboard.png** - Dashboard overview with server stats and charts
2. **servers.png** - Server list showing inventory with custom columns and tags
3. **login.png** - Login page (can be cropped/smaller)

## How to capture

1. Open ServerVault at `http://localhost:8080`
2. Use browser DevTools (F12) → Device toolbar for consistent viewport (1280x800 recommended)
3. Take screenshots using:
   - macOS: Cmd+Shift+4
   - Windows: Win+Shift+S
   - Linux: gnome-screenshot or flameshot

## Recommended dimensions

- Dashboard/Servers: 1280x720 or 1440x900
- Login: 600x400 (cropped view)

Save as PNG with descriptive names matching the README references.
